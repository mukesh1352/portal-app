# Portal AI India

Must have Bun installed!

**For installing Bun**
```bash
curl -fsSL https://bun.sh/install | bash
```
**To check if Bun is installed**
```bash
bun --version
```

**First Run**
```bash
bun install
```
**First Build is important**
```bash
bun run build
```
**Then to run the program**
```bash
bun start
```
